# VCC-clinic-App 维元诊所App V1.0
这是为北京维元诊所设计的用户端App。主要目的是为存量客户提供
1. 体检报告与治疗方案的查看
2. 服药与运动提醒
3. 在线沟通等功能。


源文件是使用Sketch+Principle制作的高保真原型


### 如何在iPhone上体验原型
App Store下载 [Principe Mirror](https://itunes.apple.com/us/app/principle-mirror/id991911319?mt=8) ，在iPhone上打开prd文件即可
